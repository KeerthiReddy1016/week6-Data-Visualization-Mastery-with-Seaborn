Project Description
This project analyzes customer purchases, sales trends, product performance, and category insights using real sales data. It includes visualizations such as line charts, box plots, violin plots, heatmaps, and interactive charts. The goal is to help understand buying behavior and business performance.
   Files Included
- analysis.ipynb – Full data analysis notebook
- sales_data.csv – Real dataset used for analysis
- dashboard visuals – Charts generated inside the notebook
   Key Analysis Performed
- Sales trends over time
- Customer segmentation
- Product and category performance
- Correlation heatmap
- Box and violin plots
- Interactive Plotly charts
- Subplots and dashboard-style layout
